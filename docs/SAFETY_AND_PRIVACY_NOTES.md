# Safety and Privacy Notes

## Public Showcase Boundaries

This repository is intended as a public showcase of architecture, workflow concepts, and applied AI systems thinking.

It should not contain private implementation details or sensitive data.

---

## Do Not Publish

Do not publish:

- API keys
- OAuth tokens
- brokerage credentials
- account IDs
- live account balances
- private portfolio data
- raw order logs
- raw brokerage responses
- account-specific trade history
- private prompts
- personal file paths
- local IP addresses
- model weights
- large raw datasets
- virtual environments
- runtime caches
- logs containing sensitive data

---

## Screenshot Redaction

Before posting screenshots, remove or crop out:

- account numbers
- order IDs
- token paths
- private local paths if sensitive
- live portfolio values if not intended for public view
- personal emails
- internal notes
- brokerage account details

---

## Financial Safety Positioning

OCIOS is a research and decision-support prototype.

It should be described as:

```text
human-in-the-loop research support
watchlist-oriented analysis
decision-support dashboard
market-intelligence workflow prototype
```

Avoid describing it as:

```text
automated trading bot
guaranteed profitable system
investment recommendation engine
production trading platform
```
